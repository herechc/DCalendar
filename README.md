# DCalendar
A datepicker for plug, not rely anymore, just javascript

# USE
``` javascript
var calendar = new DCalendar({
  id: '#demo'
})
```

# API

- id: must,the id or className for 'input';
- type: the type for plug,ex: date 
- throttle: control the slip speed
- minDate: ex: '1900-1'
- maxDate: ex: '2018-1'
